# CRUD em Angular e Spring

## Executando o Servidor

### Angular

Entre na pasta do Angular:
```
cd crud-angular
```
E rode o comando para executar o serviço:
```
npm run start
```

### Spring

Entre na pasta do Spring:
```
cd crud-spring
```
E rode o comando para executar o serviço:
```
mvn spring-boot:run
```
